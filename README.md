A modern, animated, and fully responsive portfolio website for Shiriix, a product-driven startup building advanced enterprise systems in Governance, Risk, and Compliance (GRC). The website features smooth scrolling, animated pastel backgrounds, and a professional UI for showcasing company information, services, and contact details.
Technologies Used

React.js – Frontend framework

TypeScript – Strongly-typed JavaScript

Tailwind CSS – Utility-first CSS for fast styling

React Router – Page navigation and smooth scrolling

Vite – Development server and build tool
